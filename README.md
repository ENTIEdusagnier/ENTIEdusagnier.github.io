## Markdown

Markdown es un lenguaje de marcas pensado para documentación y para ser convertido en otros formatos.

### Cabeceras

Las cabeceras se marcan con tantas almohadillas como nivel de cabecera: # es h1; ## es h2; ... asi hasta h6.
